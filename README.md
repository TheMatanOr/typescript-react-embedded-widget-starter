## Features

- Obfuscating of the code
- No css styling conflicts between the host page and the widget (with https://github.com/premasagar/cleanslate)

## Running the widget

### Install dependencies

```bash
$ yarn install
```

### Start the development server

```bash
$ yarn dev
```

### Production build

```bash
yarn build
```
